# Reportes de Ciberseguridad

Repositorio de plantillas estandarizadas en Markdown para documentación de laboratorios, auditorías, simulaciones y competiciones CTF.

## Estructura

* **EC-Council (5 Fases):** Metodología basada en Reconocimiento, Escaneo, Obtención de Acceso, Mantenimiento de Acceso y Borrado de Huellas. Orientado a auditorías de ciclo completo.
* **OffSec:** Estructura técnica orientada a explotación directa, pruebas de concepto (PoC) y captura de banderas (local.txt/proof.txt), basada en los estándares de certificación de Offensive Security.
